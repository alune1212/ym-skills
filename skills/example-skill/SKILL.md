# example-skill

## 目标

作为仓库样板，展示一个可维护的 Skill 最小实现。

## 触发条件

- 用户想参考标准 Skill 结构
- 用户需要仿照已有 Skill 创建新 Skill

## 工作流

1. 解释目录组成
2. 指出需要修改的占位内容
3. 引导维护者复制模板并补充元数据

## 输出要求

- 给出清晰结构说明
- 明确哪些文件必须更新

## 限制与注意事项

- 这是示例 Skill，不承担真实业务逻辑
